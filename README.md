# Beecrowd (URI) Challenges in Java

Este repositório reúne minhas soluções para os desafios da plataforma beecrowd (antigo URI Online Judge), desenvolvidas durante meus estudos de lógica de programação e linguagem Java.

O objetivo é fortalecer meus conhecimentos em algoritmos, estruturas de dados, resolução de problemas e boas práticas de desenvolvimento de software por meio da implementação de soluções para desafios de diferentes níveis de dificuldade.

Além do desenvolvimento em Java, este projeto também serve como ambiente de aprendizado para Git e GitHub, permitindo a prática de versionamento de código, utilização de comandos Git via terminal, criação de commits e acompanhamento da evolução do projeto ao longo do tempo.

---

## 🚀 Tecnologias e Ferramentas

* **Linguagem:** Java (versão 21)
* **IDE de Desenvolvimento:** IntelliJ IDEA
* **Plataforma de Exercícios:** [beecrowd](https://www.beecrowd.com.br/)
