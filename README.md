# 🐝 Beecrowd (URI) Challenges in Java

Este repositório foi criado com um propósito muito claro: **desenvolver, praticar e consolidar minha lógica de programação utilizando a linguagem Java**. 

Acredito que a melhor forma de dominar os fundamentos de desenvolvimento de software, estruturas de dados e algoritmos é resolvendo problemas reais. Por isso, estou enfrentando os desafios da plataforma **beecrowd** (antigo URI Online Judge) como o meu laboratório pessoal de estudos.

---

## 🚀 Tecnologias e Ferramentas

* **Linguagem:** Java (versão 21)
* **IDE de Desenvolvimento:** IntelliJ IDEA
* **Plataforma de Exercícios:** [beecrowd](https://www.beecrowd.com.br/)
